# Iconography

Here is a gallery of icons used in the project. They are pure svgs.

Icons are stored in `./src/images/icons` folder and use specific naming like `*.inline.svg` so that we can import them as components:

```javascript
import IconCheckmark from '../images/icons/checkmark-circle-dashed.inline.svg'
```
